Sketch for a simple stepper linear moved motrized hood.
controled with touch 233CH inverted (low is active). use open-stop-reverse scheme. 
auto close after 30 min timeout.

Coonctions UNO pins

DRV8825:

//   DIR  → 2

//   STEP → 3

//   EN   → 4 


//   Home switch:  LIMIT → 7  ( GND )

//   sensor 233CH: TOUCH → 8  ( LOW active )
